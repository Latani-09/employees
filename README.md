
# Usage
1. **Clone the repository:**

   ```bash
   git clone https://github.com/your-username/employee-frontend.git
   ```
2.go to project directory
   ```bash
   cd employeeFrontend
```
3.install vite
```bash
npm install -g vite
```
4.install node packages
    ```bash
    npm i
    ```
5.run
```bash
  npm run dev
